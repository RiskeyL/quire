# Quire Theme Designer Sample

This document exercises every styled element that Quire can render. Change a
theme token and watch the corresponding element update in the preview.

The body includes **bold text**, *italic text*, `inline code`, and an external
[link](https://example.com). All of these use inline-formatting rules driven by
your typography and link tokens.

## Steps

<Steps>
  <Step title="Install Quire">
    Run `npm install -g quire` to install the CLI globally.
  </Step>
  <Step title="Add a theme file">
    Create `quire.config.yaml` at the root of your docs repo.
  </Step>
  <Step title="Export to PDF">
    Run `quire export --format pdf` to produce a branded PDF.
  </Step>
</Steps>
